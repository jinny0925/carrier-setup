# Carrier Setup V5.1

## 변경사항
- 확인 범위 및 템플릿 관리에 `기타` 그룹 추가
- 프로젝트 생성 후 프로젝트 정보 수정 가능
  - 배송사명
  - 프로젝트명
  - 프로젝트 유형
  - 서비스명
  - 프로젝트 상태
  - 출발국/도착국
  - 오픈·배포 예정일
- `선택된 확인 범위`의 의미를 더 명확하게 표시
- 범위 추가 시 해당 범위의 기본 체크항목을 함께 불러올 수 있음

## 적용
1. GitHub의 기존 `index.html`을 교체
2. Commit
3. GitHub Pages 배포 후 `Ctrl + F5`
4. Firestore 규칙은 V5 규칙을 그대로 사용 가능
